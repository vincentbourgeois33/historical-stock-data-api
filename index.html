GET /financial-statements/<ticker>

Returns quarterly fundamentals for the ticker (income statement, balance sheet, cash flow). Optional ?freq=Q|A in code, default Q.

curl --request GET \
  --url "https://macrotrends-finance1.p.rapidapi.com/financial-statements/MSFT" \
  --header "x-rapidapi-key: YOUR_RAPIDAPI_KEY" \
  --header "x-rapidapi-host: macrotrends-finance1.p.rapidapi.com"

Response (compact):

{
  "income-statement": {"columns":[...],"data":[...]},
  "balance-sheet": {"columns":[...],"data":[...]},
  "cash-flow-statement": {"columns":[...],"data":[...]}
}

GET /price-history/<ticker>

Returns historical OHLCV keyed by date.

curl --request GET \
  --url "https://macrotrends-finance1.p.rapidapi.com/price-history/NVDA" \
  --header "x-rapidapi-key: YOUR_RAPIDAPI_KEY" \
  --header "x-rapidapi-host: macrotrends-finance1.p.rapidapi.com"

Response:

{
  "2024-08-09": {"Open": 482.1, "High": 490.0, "Low": 480.2, "Close": 488.3, "Volume": 32123456},
  "2024-08-08": {"Open": 475.9, "High": 484.6, "Low": 472.2, "Close": 479.0, "Volume": 28123456}
}

GET /earnings-estimates/<ticker>

Returns EPS history vs estimates, analyst consensus by period, and next earnings date. Works directly if you set a RAPIDAPI_KEY env var; otherwise pass ?rapid_key=YOUR_KEY. Optional ?compact=true for a short payload.

curl --request GET \
  --url "https://macrotrends-finance1.p.rapidapi.com/earnings-estimates/AAPL" \
  --header "x-rapidapi-key: YOUR_RAPIDAPI_KEY" \
  --header "x-rapidapi-host: macrotrends-finance1.p.rapidapi.com"

Compact shortcut:

GET https://macrotrends-finance1.p.rapidapi.com/earnings-estimates-compact/AAPL

import requests
