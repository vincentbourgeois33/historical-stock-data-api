GET /financial-statements/<ticker>

Returns quarterly fundamentals for the ticker.

curl --request GET \
  --url "https://macrotrends-finance1.p.rapidapi.com/financial-statements/MSFT" \
  --header "x-rapidapi-key: YOUR_RAPIDAPI_KEY" \
  --header "x-rapidapi-host: macrotrends-finance1.p.rapidapi.com"

Response (compact):

{
  "income-statement": {"columns":[...],"data":[...]},
  "balance-sheet": {"columns":[...],"data":[...]},
  "cash-flow-statement": {"columns":[...],"data":[...]}
}
